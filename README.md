# OpenBookAnim
打开书籍动效

## 使用说明
### 项目下的build.gradle配置
    allprojects { repositories { ... maven { url 'https://jitpack.io' } } }
### app目录下build.gradle配置
    dependencies { implementation 'com.github.Wiser-Wong:Frame:1.4.0' }
